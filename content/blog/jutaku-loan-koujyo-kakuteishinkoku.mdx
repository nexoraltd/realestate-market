マイホームを購入した方が活用できる最大の節税制度が**住宅ローン控除**（正式名称：住宅借入金等特別控除）です。最長13年間にわたって所得税・住民税が軽減されるこの制度を正しく理解することで、数百万円単位の節税につながります。

本記事では、控除額の計算方法から確定申告・年末調整の具体的な手順、2024年税制改正のポイントまで、2026年版として最新情報を完全解説します。

## 住宅ローン控除とは何か

住宅ローン控除（住宅ローン減税）とは、住宅ローンを借りてマイホームを購入・新築・リフォームした場合に、**年末のローン残高の0.7%を最大13年間、所得税から差し引ける制度**です。

### 制度の基本的な仕組み

所得税から直接控除される「税額控除」のため、節税効果が非常に大きいのが特徴です。

- **控除対象**：年末時点のローン残高
- **控除率**：0.7%（2022年度改正後）
- **控除期間**：最長13年（新築・未使用）または10年（中古）
- **控除タイプ**：所得税から直接差し引く「税額控除」

例えば、年末ローン残高が3,000万円なら3,000万円 × 0.7% = **21万円**が所得税から直接控除されます。

## 2024年税制改正の主な変更点

2024年（令和6年）の税制改正で住宅ローン控除に以下の変更が加わりました。

### 借入限度額の変更

| 住宅の種類 | 2024年入居 | 2025・2026年入居 |
|-----------|-----------|----------------|
| 長期優良住宅・低炭素住宅 | 5,000万円 | 4,500万円 |
| ZEH水準省エネ住宅 | 4,500万円 | 3,500万円 |
| 省エネ基準適合住宅 | 4,000万円 | 3,000万円 |
| その他の住宅（新築） | 3,000万円 | 0円（対象外） |
| 中古住宅 | 2,000万円 | 2,000万円 |

**重要ポイント**：2024年以降、新築で「その他の住宅」（省エネ基準を満たさない住宅）は控除対象外となりました。省エネ性能の確認が必須です。

### 所得要件の変更

2022年以降、合計所得金額が**2,000万円以下**（改正前：3,000万円以下）の方が対象です。

### 子育て・若者夫婦世帯向け特例

2024・2025年に入居する19歳未満の子を有する世帯や夫婦のどちらかが40歳未満の世帯には、借入限度額の上乗せ特例が設けられています。

- 長期優良住宅等：5,000万円（通常4,500万円）
- ZEH水準省エネ住宅：4,500万円（通常3,500万円）
- 省エネ基準適合住宅：4,000万円（通常3,000万円）

## 控除額の計算方法

### 基本の計算式

```
年間控除額 = 年末ローン残高 × 0.7%
```

ただし、上限額があります。

### 計算シミュレーション例

**【例1】長期優良住宅・2025年入居・ローン残高4,000万円**

- 借入限度額：4,500万円
- 年末残高：4,000万円（限度額内）
- 年間控除額：4,000万円 × 0.7% = **28万円**
- 13年間の最大控除総額：28万円 × 13年 = **364万円**

**【例2】中古マンション・2026年入居・ローン残高1,800万円**

- 借入限度額：2,000万円
- 年末残高：1,800万円（限度額内）
- 年間控除額：1,800万円 × 0.7% = **12.6万円**
- 10年間の最大控除総額：12.6万円 × 10年 = **126万円**

### 実際の節税額に注意

控除額は「所得税から差し引く額」です。所得税が控除額より少ない場合は、住民税からも一部控除されます（住民税控除の上限：前年課税所得の5%、最大9.75万円）。

つまり、**年収が低く所得税が少ない方は、控除額を使い切れない**ことがあります。

## 住宅ローン控除の適用要件

### 本人（借入者）に関する要件

- 合計所得金額が**2,000万円以下**であること
- 住宅取得後**6ヶ月以内に居住**し、控除を受ける年の12月31日まで引き続き居住していること
- 住宅ローンの**返済期間が10年以上**であること

### 住宅に関する要件

- **床面積が50㎡以上**（合計所得金額1,000万円以下なら40㎡以上でも可）
- 住宅ローンで購入・新築・リフォームした住宅であること
- **新築の場合**：2024年以降は省エネ基準適合住宅であること（省エネ基準を満たさない「その他住宅」は原則対象外）
- **中古住宅の場合**：昭和57年以降に建築されたもの、または現行の耐震基準に適合するもの

### 店舗併用住宅の場合

住宅部分の床面積が全体の**1/2以上**であることが条件です。また、ローン残高は住宅部分に対応する割合で計算されます。

## 確定申告の手順（1年目）

住宅ローン控除を初めて受ける年（入居した年）は、**必ず確定申告が必要**です。

### 必要書類チェックリスト

**全員必要な書類**
- [ ] 確定申告書（税務署またはe-Taxで取得）
- [ ] 住宅借入金等特別控除額の計算明細書
- [ ] 住宅取得資金に係る借入金の年末残高等証明書（金融機関から発行）
- [ ] 登記事項証明書（法務局で取得）
- [ ] 不動産売買契約書または工事請負契約書のコピー
- [ ] 源泉徴収票（給与所得者の場合）

**新築・省エネ住宅の場合に追加**
- [ ] 長期優良住宅認定通知書（長期優良住宅の場合）
- [ ] 低炭素建築物認定通知書（低炭素住宅の場合）
- [ ] 建設住宅性能評価書または住宅省エネルギー性能証明書

**中古住宅の場合に追加**
- [ ] 耐震基準適合証明書または既存住宅売買瑕疵保険の保険証券（昭和57年以前の物件）

### 申告期限

購入した年の翌年、2月16日〜3月15日が確定申告の期限です。

### e-Taxでの申告手順

1. 国税庁のe-Taxサイトにアクセス
2. 「確定申告書等作成コーナー」から申告書を作成
3. 住宅借入金等特別控除の計算明細書を入力
4. 必要書類を添付（スキャン・写真でOK）
5. マイナンバーカードで電子署名して送信

税務署への持参や郵送も可能です。

## 年末調整の手順（2年目以降）

確定申告で初年度の手続きを済ませると、2年目以降は**勤務先の年末調整**で控除を受けられます。

### 年末調整の流れ

1. **毎年10〜11月頃**：金融機関から「年末残高等証明書」が届く
2. **11月〜12月**：勤務先から「給与所得者の(特定増改築等)住宅借入金等特別控除申告書」の用紙が配布される
3. **用紙に必要事項を記入**：年末残高・借入限度額・控除額を記入
4. **「年末残高等証明書」を添付して勤務先に提出**
5. 勤務先が年末調整で控除を計算・還付

### 年末調整で控除しきれない場合

年末調整で控除しきれなかった場合（所得税が控除額より少ない場合）、住民税からの控除は自動的に行われます。別途の手続きは不要です。

## 住宅ローン控除が適用されない・減額されるケース

### 適用除外のケース

- **転勤・単身赴任**：住宅から転出した場合は適用外（ただし転勤期間中は一時的に停止し、戻ってきたら再開できるケースあり）
- **別荘・セカンドハウス**：居住の用に供しない場合は対象外
- **賃貸に出した場合**：住宅の一部を賃貸にすると、その部分は控除対象外
- **親族から購入した場合**：直系血族間の売買は対象外
- **生計を一にする者から購入した場合**：対象外

### 繰り上げ返済の注意

繰り上げ返済を行って返済期間が10年未満になると、控除が受けられなくなります。控除期間中の繰り上げ返済は慎重に検討しましょう。

## 住宅ローン控除と併用できる制度

### すまい給付金（終了）

すまい給付金は2021年12月末で終了しました。2022年以降の取得には適用されません。

### 住宅取得等資金の贈与の非課税特例

親・祖父母から住宅取得資金の援助を受けた場合、一定額まで贈与税が非課税になります。住宅ローン控除と**併用可能**です。

- 省エネ等住宅：1,000万円まで非課税
- 一般住宅：500万円まで非課税

### 固定資産税の減額

新築住宅は取得後3〜5年間、固定資産税が1/2に減額されます（長期優良住宅は5〜7年）。住宅ローン控除と**併用可能**です。

## よくある質問（FAQ）

### Q1：フラット35でも住宅ローン控除は使えますか？

はい、フラット35（住宅金融支援機構の固定金利型住宅ローン）も住宅ローン控除の対象です。返済期間が10年以上であることが条件です。

### Q2：ペアローンの場合はどうなりますか？

夫婦それぞれがローンを組むペアローンの場合、**それぞれが住宅ローン控除を受けられます**。ただし、各自のローン残高が控除の基準になります。

### Q3：頭金を多く入れると控除額が減りますか？

はい、頭金を多く入れると年末ローン残高が少なくなるため、控除額も小さくなります。節税の観点からは頭金を少なくしてローン残高を多くする方が有利ですが、金利負担も増えるためバランスを考慮してください。

### Q4：住み替えで新旧両方の住宅ローン控除は受けられますか？

前の住宅（売却済み）のローン控除と、新しい住宅のローン控除の**重複は原則不可**です。また、マイホームの3,000万円特別控除（譲渡所得の特例）と住宅ローン控除は、同一年に両方は使えません。

### Q5：確定申告を忘れた場合はどうなりますか？

還付申告の場合は**5年以内であれば遡って申告可能**です。入居した年に申告し忘れた場合でも、5年以内なら確定申告できます。
