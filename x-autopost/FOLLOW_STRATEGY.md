# X自動フォロー現状と代替戦略

## 現状分析

### API制限の壁

**問題:** X API Free Tier で GET `/2/tweets/search/recent` が 403 Forbidden

```
Error: The request has been denied
Reason: Endpoint not available for the Free tier
```

- **原因:** Twitter API v2 の Search Endpoints はすべて Academic Research以上が必須
- **コスト:** 有料プラン $100+/月（検索API単独）
- **代替案コスト:** 実装なし（ブラウザ手動フォロー + 営業コメント）

---

## 代替戦略（4段階アプローチ）

### 代替策1: 手動ターゲット検索 + フォロー（1日5分）

**実行方法:**
1. Xブラウザの検索機能で キーワード+ユーザーフィルタ検索
2. ターゲットカテゴリ別に1日あたり5検索
3. 各検索で上位10-15ユーザーをフォロー
4. **1日50人程度** の安定フォロー

**ターゲットキーワード:**
```
不動産投資 / FIRE / 資産形成 / マイホーム / 住宅ローン / 注文住宅
```

**時間効率:**
- 1検索 (クリック→スクロール→フォロー) : 30秒/人
- 10人フォロー : 5分
- 1日50人 : 25分（週3日 = 75分/週）

**効果:**
- ターゲット層への直接露出
- フォローバック率 15-25%（業界平均）
- 月間純増 50-80人（フォロワーロス除外）

**KPI:** 月間フォロワー + 100人

### 代替策2: リプライ営業（提供価値型）

**コンセプト:** スパムではなく「相場データの活用提案」

**実行方法:**
1. ターゲットキーワード + 過去3日 のツイートを検索
2. 不動産購入・投資相談系のツイートを抽出
3. 以下パターンで「相場データ活用法」をリプライ

**リプライテンプレート:**

```
【相場確認例】
「この地域のマンション、本当にお得？」
→ 過去20年の取引データを見ると、坪単価で平均+3.2%
→ あなたの物件候補もデータで検証してみませんか？
👇 https://market.next-aura.com/search
```

```
【投資利回り分析例】
「表面利回り5%で購入検討中」
→ 実質利回りは管理費・空室・税金を計算すると2-3%
→ 地域データで「平均利回り」を確認することをお勧めします
👇 https://market.next-aura.com/blog/fudousan-toshi-rimawari
```

```
【ローン返済額例】
「月々の返済額が心配」
→ 金利0.3% vs 1.5% では月額差 15,000円
→ 35年だと630万円の差です
→ 無料で返済シミュレーション可能です
👇 https://market.next-aura.com/tools/loan-simulator
```

**実行スケジュール:**
- 1日あたり 10件のリプライ営業
- 投稿時間 : 朝（8-10時）、昼（12-13時）、夜（19-21時）
- 各リプライ後 30分以内にフォロー

**効果測定:**
- クリックスルー率（CTR） : 1-3%
- メルマガ登録 : 月間 +15人程度
- 継続リプライされたツイートへのプロフ訪問 : 2-5件/月

**KPI:** ウェブサイトクリック数 月間 +30

### 代替策3: コンテンツ品質による自然フォロー獲得

**実装済み項目:**
- ✅ ハッシュタグ追加（#不動産相場 #マイホーム #不動産投資）
- ✅ データファクト型ツイート（通説検証アプローチ）
- ✅ 投稿時刻最適化（朝8:13 / 昼12:30 / 夜19:45）

**実装予定項目:**
1. **リツイート買収戦略**
   - インフルエンサー（フォロワー10k+）へのDM : 「この投稿共有していただけますか？」
   - 対象 : 不動産投資・マイホーム系インフルエンサー 5-10人/月

2. **スレッド化による滞在時間増加**
   - 単一ツイート（280字）→ スレッド化（3-4ツイート）
   - 効果 : リツイート率 +40%、コメント率 +60%

3. **ユーザー参加型ポストの実験**
   ```
   例：「あなたのエリアのマンション相場は？」
   　→ 回答形式でリプライさせる
   　→ リプライユーザーをフォロー
   ```

**効果:**
- 自然フォロー（コンテンツ品質由来） : 月間 +30-50人
- エンゲージメント率 : +25%

**KPI:** 「いいね」3件/ツイート以上（現在1-2件）

## リスク管理

### リスク1: X API の利用規約違反

**対策:**
- リプライ営業は「営業」ではなく「情報提供」として実施
- スパムフォローにならないよう、フォロー後 3日以内にフォローバック確認
- アンフォロー機能で非ファローバック者を自動削除（14日ルール）

### リスク2: エンゲージメント不足

**対策:**
- テンプレート A/B テスト（2週間ごと）
- 返信がついたツイートのテーマを分析
- 「通説検証」型は確実にエンゲージメントが高いため、継続

### リスク3: マンパワー不足

**対策:**
- リプライ営業の自動化（今後実施）
- テンプレート化で対応時間を削減
- 月1回のバッチ実施に変更可能
